# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-09

### Added
- Initial release
- Speaker diarization using pyannote.audio
- Transcription using OpenAI Whisper
- Multi-LLM support (OpenAI, Anthropic, OpenRouter)
- AI-powered analysis: summaries, action items, Q&A
- Docker containerization
- Comprehensive documentation

### Features
- Supports .m4a, .mp3, .wav, .mp4 audio formats
- Privacy-first: local processing, optional cloud analysis
- Configurable via YAML
- Interactive Q&A mode with transcripts
